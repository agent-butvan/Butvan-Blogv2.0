# 变更日志 (CHANGELOG)

本文档记录 **Butvan Blog 2.0** 的全部版本更新明细与发版履历，遵循 [Semantic Versioning 2.0.0](https://semver.org/lang/zh-CN/) 版本规范。

---

## v2.1.0

- **发布日期 (UTC)**：2026-07-24
- **对比基线**：v2.0.0
- **发布通道**：Stable

v2.1.0 集中完善了内容渲染与编辑器体验，增加了全新的图片交互与富文本/代码调色盘功能，并完成了全站版本管理控制。

### 内容渲染与编辑器体验

- **行内代码高亮与配色**：前后台行内代码实现 100% 高保真浅紫微边框配色对齐，增加调色盘支持自由设置字色与高亮背景。
- **图片无法显示卡片与悬浮编辑**：图片悬浮显示编辑图标，点击弹窗（Modal）可配置图片 URL 地址、描述及删除；图片加载失败时自动展示提示警告卡片与替换操作。
- **智能粘贴图片**：支持直接粘贴可访问的网络图片 URL 自动渲染为图片节点。

### 排版与阅读体验

- **消除引用留白**：修复前台文章详情页 `>` 引用块底部过大留白与竖线拉长问题，使竖线与文本内容高度精准契合。
- **行高与段落间距**：优化前台文章正文行高至 1.72，收紧段落下边距与标题间距，提升沉浸式阅读品质。

### 备份恢复与工程优化

- 全站建立语义化版本（SemVer）控制管理体系与 GitHub Release 自动发布工作流。
- 同步更新项目结构与 `DIRECTORY.md` 架构归档。

### 升级说明

- 本版本不包含新的数据库迁移，按现有升级流程更新到 2.1.0 即可。
